# Project Setup Instructions

> **⚠️ Important Check**: It seems Node.js is not currently installed or available in your system PATH.
> To run this project, you must install [Node.js (LTS version)](https://nodejs.org/) first.

## Prerequisities
- Node.js v18+
- PostgreSQL
- Redis

## Getting Started

### 1. Frontend (Next.js)
```bash
cd frontend
npm install
npm run dev
```

### 2. Backend (NestJS)
```bash
cd backend
npm install
npm run start:dev
```
